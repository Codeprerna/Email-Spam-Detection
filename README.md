## 🧠 Overview

This project focuses on building an **Email Spam Detection System** using **Natural Language Processing (NLP)** and **Machine Learning** techniques. It classifies incoming emails/messages as **Spam** or **Not Spam** by analyzing textual patterns and linguistic features.

---

## ✨ Key Highlights

✔️ End-to-end ML pipeline
✔️ Text preprocessing & cleaning
✔️ Feature extraction using TF-IDF
✔️ Spam classification using Naive Bayes
✔️ Performance evaluation with metrics
✔️ Easily extendable to real-world applications
